# Student Assistant AI

An AI-powered student assistant application designed to help students with homework, assignments, projects, and studying. This app acts as a comprehensive academic companion, featuring natural language chat, study tools, reminders, and personalization.

## Features

*   **AI Chatbot**: Advanced conversational interface for homework help, explanations, and research.
*   **Study Sets**: Create flashcards, summaries, and notes for effective learning.
*   **Smart Reminders**: Keep track of assignments, exams, and study sessions with customizable notifications.
*   **Personalization**: Tailor the app to your grade level, interests, and preferred communication style.
*   **Dark Theme**: A beautiful dark mode interface with luxurious burgundy, black, and gold accents.

## Tech Stack

*   **Framework**: Flutter
*   **UI/UX**: Material Design 3 with custom theming
*   **State Management**: Stateful widgets (ready for Riverpod/Provider scale-up)

## Getting Started

1.  Ensure you have Flutter installed.
2.  Clone this repository.
3.  Run `flutter pub get` to install dependencies.
4.  Run `flutter run` to launch the app on your preferred device.
